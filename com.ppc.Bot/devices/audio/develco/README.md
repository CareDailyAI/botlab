# Develco Audio Assistant Device Class Information

This device is a BLE audio device that utilizes VOIP services provided by the Develco Squidlink Gateway

The device provides 3 buttons, which get measured either as short press (1) or long press (2).  Note, the buttons do not provide a value describing an "off" status.

VoIP call statuses and other parameters are provided by the proxy Develco Squidlink Gateway device.
